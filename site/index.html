Project Enterprise Connect

A minimal Amazon Connect contact center with IVR, queue routing, real-time metrics, and optional Lambda → DynamoDB audit.

What’s in this build

  • IVR menu (DTMF): 1 = Rentals, 2 = Fleet
  • Routing to SupportQueue → agent softphone (CCP)
  • Real-time metrics for operations
  • Optional audit hook via Lambda → DynamoDB
